# Dataset

## Source

Retail Sales Dataset (Kaggle)

Link: https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset

## Description

This dataset contains transactional retail sales data along with customer demographics, product information, and purchase dates. The dataset was used to analyze sales performance, customer purchasing behavior, and product category trends.

## Data Includes

- Transactional information
- Customer demographics
- Product categories
- Sales quantity
- Sales revenue
- Purchase dates

## Usage

The dataset was imported into Google Bigquery, cleaned and transformed using SQL, and used to create Tableau dashboards analyzing revenue trends, customer segments, and product performance.
